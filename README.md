# Fresh Squeeze 🍊

A fresh juice ordering website for a local juice shop — built to give customers a clean, modern way to browse the menu and place orders online.

---

## What It Does

Fresh Squeeze is a front-end website for a real juice shop based in Sangli, Maharashtra.
Customers can browse the menu, see prices, and place orders — all from a clean, mobile-friendly interface.

---

## Features

- **Juice Menu** — Cards for each juice with name, description, and price
- **Order Flow** — Direct links to the order page from every juice card
- **About Section** — Highlights what makes the shop stand out (organic, fresh, made with care)
- **Contact & Location** — Address, phone number, and working hours
- **Responsive Design** — Works on mobile and desktop
